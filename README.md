# Flappy Bird Clone

A simple implementation of the classic Flappy Bird game built with Python and Pygame. This project features straightforward mechanics, allowing players to navigate a bird through a series of pipes while avoiding obstacles.

## Features

- Simple and addictive gameplay
- Basic scoring system
- Customizable graphics and sounds
- Responsive controls
- Scalable architecture for future enhancements

## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/flappy-bird.git
   cd flappy-bird
   ```

2. **Set up a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install the required packages:**
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To start the game, run the following command:

```bash
python main.py
```

Use the spacebar to make the bird flap and navigate through the pipes. Aim for the highest score!
